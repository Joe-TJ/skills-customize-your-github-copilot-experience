# 📘 Assignment: Functions and Dictionaries in Python

## 🎯 Objective

Build a simple contact book using Python functions and dictionaries. In this assignment, you will practice organizing your code into reusable functions and storing structured data in a dictionary.

## 📝 Tasks

### 🛠️ Create the Contact Book Structure

#### Description
Set up the main data structure for the program and create a function that displays all saved contacts in a clear format.

#### Requirements
Completed program should:

- Use a dictionary to store contact information.
- Store at least a name and phone number for each contact.
- Create a function that prints all contacts in the contact book.
- Show a helpful message when no contacts have been added yet.


### 🛠️ Add and Search for Contacts

#### Description
Write functions that allow the user to add a new contact and search for an existing contact by name.

#### Requirements
Completed program should:

- Create a function that adds a new contact to the dictionary.
- Ask the user for the contact name and phone number.
- Create a function that searches for a contact by name.
- Display the matching contact information when a contact is found.
- Display a clear message when a contact does not exist.


### 🛠️ Build a Menu with Contact Actions

#### Description
Combine your functions into a simple menu-driven program so the user can choose what action to perform.

#### Requirements
Completed program should:

- Display a menu with options such as view, add, search, and quit.
- Call the correct function based on the user's menu choice.
- Repeat until the user chooses to quit the program.
- Handle invalid menu choices without crashing.
